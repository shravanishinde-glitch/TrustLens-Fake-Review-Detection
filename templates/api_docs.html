TrustLens AI API Documentation

API Overview

Welcome to the TrustLens AI API. Our REST API allows you to programmatically analyze reviews for authenticity and trust scores.

Base URL:
https://api.trustlens.ai/v1
Authentication:
Bearer Token (API Key)
Response Format:
JSON
Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
Endpoints
POST /analyze

Analyze a review for authenticity.

Request Body: 
{
  "review_text": "This product is amazing! Best purchase ever!",
  "product_id": "optional_product_identifier"
}
Response: 
{
  "prediction": "Real",
  "confidence": 0.92,
  "trust_score": 87.5
}
GET /reviews

Get analyzed reviews for your products.

Query Parameters:
  • product_id (optional): Filter by product
  • limit (optional): Number of results (max 100)
  • offset (optional): Pagination offset
Response: 
{
  "reviews": [
    {
      "id": "review_123",
      "text": "Great product!",
      "prediction": "Real",
      "confidence": 0.89,
      "trust_score": 85.2,
      "timestamp": "2024-03-09T10:30:00Z"
    }
  ],
  "total": 150,
  "limit": 10,
  "offset": 0
}
GET /analytics

Get analytics data for your products.

Response: 
{
  "total_reviews": 150,
  "real_reviews": 135,
  "fake_reviews": 15,
  "average_trust_score": 82.3,
  "products": [
    {
      "id": "prod_123",
      "name": "iPhone 15",
      "review_count": 50,
      "fake_percentage": 8.0
    }
  ]
}
Rate Limits
  • Analyze endpoint: 100 requests per minute
  • Get reviews: 500 requests per minute
  • Get analytics: 100 requests per minute

Rate limits reset every minute. Exceeding limits returns HTTP 429 (Too Many Requests).

Code Examples
Python
import requests

api_key = "your_api_key_here"
headers = {"Authorization": f"Bearer {api_key}"}

# Analyze a review
response = requests.post("https://api.trustlens.ai/v1/analyze", 
                        json={"review_text": "Amazing product!"}, 
                        headers=headers)
print(response.json())
JavaScript
const apiKey = "your_api_key_here";
const headers = { "Authorization": `Bearer ${apiKey}` };

fetch("https://api.trustlens.ai/v1/analyze", {
  method: "POST",
  headers: {
    ...headers,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    review_text: "Amazing product!"
  })
})
.then(response => response.json())
.then(data => console.log(data));
